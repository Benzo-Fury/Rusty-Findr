# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-09

### Added

- Web frontend with React SPA served from embedded assets
- Profile button with dropdown menu
- Owned title and library dialog
- Cross-compiled binaries for Linux (x86_64, arm64), macOS (Intel, Apple Silicon), and Windows (x86_64)
- GitHub Actions CI for automated release builds
